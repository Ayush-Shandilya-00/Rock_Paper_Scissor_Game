# 🪨📰✂️ Rock, Paper, Scissors Game

A fun, interactive terminal-based Rock, Paper, Scissors game built using Python where you play against the computer.

## ✨ Features
* **Randomized AI**: The computer picks options randomly using Python's `random` module.
* **Score Keeping**: Tracks player and computer wins during the session.
* **Input Validation**: Keeps playing seamlessly even if a user types an invalid choice.

## 🚀 How to Run the Project
1. Open your terminal in the project folder.
2. Run the command:
   ```bash
   python main.py
   ```
